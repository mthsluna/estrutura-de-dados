# Estrutura de Dados

Repositório com estudos e implementações de estruturas de dados em C++.

---

## Conteúdo

- Listas, pilhas e filas
- Árvores (binária, AVL, etc.)
- Grafos
- Tabelas hash
- Algoritmos de ordenação e busca

---

## Tecnologias

- C++
- g++ / MinGW

---

## Como Executar

```bash
git clone https://github.com/seu-usuario/estrutura-de-dados.git
cd estrutura-de-dados

# Compilar um arquivo
g++ arquivo.cpp -o saida

# Executar
./saida
```
