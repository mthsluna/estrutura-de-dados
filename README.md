## 🍬 O que é Estrutura de Dados?
Estrutura de Dados é tipo **uma caixinha mágica** onde guardamos informações de um jeito organizado.  
Dependendo da estrutura, você consegue achar, adicionar ou remover dados **super rápido**! ⚡

> "Não é só guardar dados, é saber como brincar com eles!" 🎈

---

## 🐱 Principais Estruturas

### 1. Arrays (Vetores) 🌈
- Uma fileirinha de caixinhas ⬜⬜⬜
- Cada caixinha tem um **índice**
- Boa pra acessar rápido, mas **não é tão flexível** pra inserir ou remover

Exemplo:  
```txt
[🐶, 🐱, 🐭, 🐹]
índices: 0   1   2   3
